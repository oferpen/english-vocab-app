# מדריך התחלה מהירה

## התקנה מהירה

```bash
# 1. התקן תלויות
npm install

# 2. הגדר מסד נתונים
npx prisma generate
npx prisma db push

# 3. טען נתונים
npm run db:seed

# 4. הפעל
npm run dev
```

## כניסה ראשונה

1. פתח `http://localhost:3000`
2. לחץ על "הורים" בחלק העליון
3. הגדר PIN (ברירת מחדל לאחר seed: `1234`)
4. הוסף ילד בטאב "ילדים"
5. צור תוכנית יומית בטאב "תוכנית יומית"
6. לחץ על "Starter Pack" ליצירת תוכנית מהירה

## מבנה הנתונים

- **ParentAccount**: חשבון הורה יחיד עם PIN
- **ChildProfile**: פרופילי ילדים
- **Word**: מילים באנגלית עם תרגום עברי
- **DailyPlan**: תוכניות יומיות למילים
- **Progress**: התקדמות לכל ילד ולכל מילה
- **QuizAttempt**: ניסיונות חידון
- **MissionState**: מצב משימות יומיות/שבועיות
- **LevelState**: רמה ו-XP לכל ילד

## נתוני התחלה

לאחר seed, המערכת כוללת:
- PIN: `1234` (שנה מיד!)
- ילד ברירת מחדל: "ילד/ה"
- 60+ מילים בקטגוריות: Home, School, Animals, Colors, Food

## תכונות עיקריות

### ממשק ילדים
- **ללמוד**: למידת מילים עם הגייה (TTS)
- **חידון**: שאלות מרובות בחירה
- **התקדמות**: רמות, XP, רצף ימים, משימות

### פאנל הורים
- **ילדים**: ניהול פרופילים
- **מילים**: CRUD מלא
- **תוכנית יומית**: הקצאת מילים לתאריכים
- **דשבורד**: סטטיסטיקות והתקדמות
- **הגדרות**: התאמה אישית

## הערות

- כל הממשק בעברית RTL
- עיצוב מובייל-פירסט
- PIN מאוחסן כגיבוב (bcrypt)
- ללא התחברות חיצונית
