# אפליקציית לימוד אנגלית לילדים

אפליקציית ווב מובייל-פירסט ללימוד אוצר מילים באנגלית לילדים, עם ממשק בעברית (RTL).

## טכנולוגיות

- **Next.js 14** (App Router) + TypeScript
- **Tailwind CSS** לעיצוב
- **Prisma** + SQLite למסד נתונים
- **Zustand** לניהול מצב לקוח
- **Server Actions** לפעולות נתונים
- **Web Speech API** להגייה (TTS)

## התקנה והרצה

### דרישות מוקדמות

- Node.js 18+ 
- npm או yarn

### שלבי התקנה

1. התקן תלויות:
```bash
npm install
```

2. הגדר את מסד הנתונים:
```bash
npx prisma generate
npx prisma db push
```

3. טען נתוני התחלה (seed):
```bash
npm run db:seed
```

4. הפעל את השרת:
```bash
npm run dev
```

5. פתח בדפדפן:
```
http://localhost:3000
```

## שימוש באפליקציה

### ממשק ילדים

האפליקציה פועלת תמיד בהקשר של ילד פעיל אחד. הממשק כולל:

#### ללמוד (`/learn`)
- **היום**: מילים שהוקצו ליום הנוכחי
- **עוד ללמוד**: המשך למידה מעבר לתוכנית היומית

#### חידון (`/quiz`)
- **חידון היום**: חידון על המילים של היום
- **תרגול נוסף**: תרגול נוסף לפי קטגוריות או מילים שצריכות חיזוק

#### התקדמות (`/progress`)
- רמה ו-XP
- רצף ימים
- סטטיסטיקות
- משימות יומיות ושבועיות

### פאנל הורים (`/parent`)

פאנל הורים מוגן ב-PIN עם 5 טאבים:

#### 1. ילדים
- הוסף, ערוך ומחק פרופילי ילדים
- הגדר ילד פעיל

#### 2. ניהול מילים
- CRUD מלא למילים
- הוסף מילים באנגלית עם תרגום עברי
- קטגוריות, רמת קושי, דוגמאות

#### 3. תוכנית יומית
- בחר תאריך וילד
- הקצה מילים לתוכנית יומית
- **Starter Pack**: יצירה אוטומטית של תוכנית עם 10 מילים מ-Home + School
- **יצירה אוטומטית**: יצירת תוכנית לפי פילטרים (קושי, קטגוריה, מילים שלא נראו)

#### 4. דשבורד התקדמות
- סטטיסטיקות לכל ילד
- טבלת התקדמות לפי מילה
- שליטה, ניסיונות חידון, מילים שצריכות חיזוק

#### 5. הגדרות
- סוגי שאלות בחידון (אנגלית→עברית, עברית→אנגלית, שמיעה→אנגלית)
- אורך חידון
- אסטרטגיית למידה נוספת
- כלל רצף ימים
- עוצמת פרסים

## מערכת שליטה (Mastery)

כל מילה מקבלת ציון שליטה (0-100%) המבוסס על:
- מספר הפעמים שנראתה בלמידה
- אחוז נכונות בחידונים
- האם נדרש חיזוק

מילה נחשבת "שולטת" כאשר ציון השליטה ≥ 80%.

## מערכת משימות

### משימות יומיות
- למד מילים היום
- השלם חידון

### משימות שבועיות
- למד 5 מילים
- השלם חידון 4 ימים

## מערכת רמות ו-XP

ילדים מתקדמים ב-10 רמות (1-10) על בסיס:
- מילים שולטות: 10 XP כל אחת
- השלמת משימות: 50 XP כל אחת
- רצף ימים: 5 XP ליום

## רצף ימים (Streak)

רצף ימים מתעדכן לפי כלל שנקבע בהגדרות:
- **רק למידה**: השלמת למידה יומית
- **רק חידון**: השלמת חידון יומי
- **אחד מהם**: השלמת אחד מהשניים
- **שניהם**: השלמת שניהם

## נתוני התחלה (Seed)

לאחר הרצת `npm run db:seed`, המערכת כוללת:
- חשבון הורה עם PIN: `1234`

- 60+ מילים מתחילות בקטגוריות:
  - בית (Home) - 22 מילים
  - בית ספר (School) - 22 מילים
  - בעלי חיים (Animals) - 8 מילים
  - צבעים (Colors) - 8 מילים
  - אוכל (Food) - 7 מילים


## הערות חשובות

1. **אבטחה**: PIN מאוחסן כגיבוב (hash) באמצעות bcrypt
2. **RTL**: כל הממשק בעברית עם תמיכה מלאה ב-RTL
3. **מובייל-פירסט**: עיצוב מותאם למסכים קטנים
4. **ללא פרסומות**: האפליקציה נקייה מפרסומות
5. **ללא התחברות חיצונית**: אין צורך ב-Google OAuth או התחברות אחרת
